Covid19 Data Analysis Notebook¶

Objective:¶

In this Covid19 Data Analysis NOtebook, we will try to explore data available from open source to identify how Covid19 is impacting happyness in a country.

Here we will use below two datasets:

  1. Covid19 dataset published by John Hopkin's university related to cumulative data of confirmed cases in a country.
  2. World happiness index dataset from United Nations

Key objectives of this tutorial notebook are as follows:

  1. How to Load data from a csv file
  2. How to visualize dataframe and each of the variable present in dataset
